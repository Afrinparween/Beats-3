# 🎧 Beats 3 Product Landing Page
A modern dark-themed product landing page inspired by Beats headphones.
This project focuses on building a visually appealing product showcase website using HTML and CSS, featuring a premium UI layout, product sections, and responsive design structure.

The goal of this project was to practice real-world landing page design, layout structuring, and modern UI styling similar to professional product websites.

🚀 Features

✔ Modern Dark UI Design

✔ Product Hero Section with Call-to-Action

✔ Clean Navigation Bar

✔ Product Overview Section

✔ Specifications Section

✔ Product Style Variants Display

✔ Brand Integration Section (Apple, Spotify, Amazon, YouTube)

✔ Structured Footer Layout

🧩 Sections Included
1️⃣ Hero Section

Main Beats 3 headphones display

Product overview description

Price and "Add to Bag" button

Clean typography and spacing

2️⃣ Case Section

Displays the headphone case with product highlight.

3️⃣ Style Selection

Users can explore different headphone color styles.

4️⃣ Specs Section

Technical specifications such as:

Bluetooth connectivity

Battery duration

Fast charging support

Microphone support

5️⃣ Brand Compatibility

Shows integration icons such as:

Apple

Spotify

Amazon

YouTube

6️⃣ Footer

Includes product and support links for navigation.
